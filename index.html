现代动画库

悬浮动画

一个平滑优雅的上下浮动效果，为界面增添生动的视觉层次。适用于需要突出显示的UI元素，让用户体验更加自然流畅。

完美适用于：

  • 功能引导图标
  • 重要操作按钮
  • 产品展示卡片

CSS 代码：

@keyframes float {
    0%,
    100% {
      transform: translateY(0);
    }

    50% {
      transform: translateY(-20px);
    }
  }

  .float {
    animation: float 3s ease-in-out infinite;
  }

HTML 使用：

<div class="float">浮动的元素</div>
